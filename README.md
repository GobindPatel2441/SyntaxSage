# SyntaxSage

A web-based AI coding assistant built with Flask and LangChain, designed to assist developers by answering queries, generating code, and debugging. The application utilizes Groq for accelerated language model inference.

## Features

- **Code Generation**: Easily generate code snippets and entire functions based on natural language descriptions.
- **Error Solving**: Identify and debug errors in your codebase with intelligent suggestions and fixes.
- **Logic Explanation**: Understand complex logic and undocumented code snippets with clear, natural language explanations.

## Technologies Used

- **Backend**: Flask
- **LLM Orchestration**: LangChain
- **Inference**: Groq API

## Setup
Install requirements:
```bash
pip install -r req.txt
```

Run the application:
```bash
python run.py
```
